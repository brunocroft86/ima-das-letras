#  Imã das letras
Uma criação de códigos no qual temos uma tela repleta de letras,\
em que você vai passando o mouse e o cursor vai atraindo as letras.\
É algo bem legal e vale a pena você conferir.\
Caso abra no celular, você toca em algum lugar da tela e as letras<br/> vão
indo para seu dedo como se ele fosse um imã atraindo as letras.

## Foi usado:
- HTML
- JavaScript
- JavaScript Object Notation (JSON)
